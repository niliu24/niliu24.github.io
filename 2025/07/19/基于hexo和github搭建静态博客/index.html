基于hexo和github搭建静态博客

环境准备

  1. 安装git
  2. 安装node.js
  3. 安装hexo框架：npm install hexo-cli -g

配置github仓库

  1. 创建github仓库：登录GitHub，新建名为用户名.github.io 的仓库（如用户名为a，则仓库名必须为a.github.io ）
  2. 配置ssh密钥

本地博客初始化

  1. 初始化本地博客：hexo init <folder>
  2. 生成博客：hexo g
  3. 启动博客进行预览：hexo s
