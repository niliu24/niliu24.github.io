# 搭建博客

# 准备环境

  1. 安装 git

  2. 安装 node.js ：使用 nvm 安装 nodejs 的步骤如下：

    1. 安装 nvm ： curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

    2. 将安装成功后的配置信息添加到 shell 配置文件：

    3. 重启加载 shell 配置文件

    4. 使用 nvm ls-remote 查看可用的 node 版本

    5. 使用 nvm install x.x.x 安装指定版本 node

    6. 因为网络原因， npm 安装包时速度很慢，可以替换为 cnpm: npm install -g cnpm

  3. 安装 hexo 框架： npm install hexo-cli -g

# 配置 github 仓库

  1. 创建 github 仓库：登录 GitHub，新建名为 <username>.github.io 的仓库（如用户名为 a，则仓库名必须为 a.github.io ）
  2. 配置 ssh 密钥：将本地 .ssh 目录下的 id_rsa.pub 文件内容添加到 github 的 ssh 密钥中

# 初始化本地博客

  1. 初始化本地博客： hexo init <folder>
  2. 生成博客： hexo g
  3. 启动博客进行预览： hexo s

# 部署到 github

  • 安装部署插件：在项目目录执行 npm install hexo-deployer-git --save 安装 Git 部署插件
